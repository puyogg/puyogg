## Development

Make sure you're using these Node and pnpm version!!!

- Node v20
- pnpm v8 (try `npm install -g pnpm@8`)

Copy env files:

```sh
./scripts/init-env.sh
```

Log into GitHub Registry with classic token:

```sh
pnpm login --scope=@puyogg --auth-type=legacy --registry=https://npm.pkg.github.com
```
